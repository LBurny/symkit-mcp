## 回报漏洞

如果你发现安全漏洞，请**不要**公开在 Issues 中回报。

### 回报方式

1. 发送邮件至：[security@example.com]
2. 或使用 GitHub 的私人漏洞回报功能

### 回报内容

请包含以下信息：

- 漏洞描述
- 重现步骤
- 可能的影响范围
- 建议的修复方式（如果有）

### 回应时程

- 48 小时内确认收到回报
- 7 天内初步评估
- 90 天内修复（视严重程度调整）

### 感谢

我们感谢所有负责任地回报安全问题的研究人员。
